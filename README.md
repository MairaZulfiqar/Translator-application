# Safa's Translator 🌍🎙️

A premium, real-time voice translation application that seamlessly converts speech between **English** and **Finnish**.

## 🚀 Overview

**Safa's Translator** is designed for low-latency, bi-directional communication. Using the browser's native Speech Recognition capabilities and the MyMemory Translation API, it provides an instant text-based translation of your spoken words.

## ✨ Key Features

- **Live Voice Input**: Real-time speech-to-text transcription.
- **Finnish ↔️ English**: Easy toggle to switch between translation directions.
- **Premium Design**: Modern, glassmorphic UI with animated feedback and responsive layout.
- **Low Latency**: Optimized to provide translations immediately after a sentence is completed.
- **No Installation**: Runs entirely in the browser.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom design system with glassmorphism and animations.
- **JavaScript (ES6+)**: Core application logic.
- **Web Speech API**: Handles real-time voice recognition.
- **MyMemory API**: Powers the translation engine.

## 📖 How to Use

1. **Open the App**: Simply open `index.html` in a modern web browser (Chrome or Edge recommended).
2. **Access Microphone**: Click **Start Listening** and grant the browser permission to use your microphone.
3. **Set Language**: Use the toggle switch to choose your source and target language:
   - **Left (Default)**: English to Finnish.
   - **Right (Toggled)**: Finnish to English.
4. **Speak**: Start talking! Your speech will appear in the "Original Speech" box, and the translation will appear in the "Translation" box shortly after.

## 📝 Requirements

- An active internet connection (for the translation API).
- A microphone.
- A supported browser (Chrome, Edge, or Safari).
